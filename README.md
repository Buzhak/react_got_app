# react_got_app

Проект базы знаний по Игре Престолов.
Сделан с использованием фреймворка React, в целях обучения.

## Запуск проекта

1. Клонируем репозиторий
  ```
  git clone git@github.com:Buzhak/react_got_app.git
  ```
2. Устанавливаем зависимости.
  ```
  npm install
  ```
3. Запускаем проект.
  ```
  npm start
  ```


## Технологии
* JS
* HTML
* CSS
* REACT
* JSX
